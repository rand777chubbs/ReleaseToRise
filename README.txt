Publish (browser only):
1) Open your repo → Add file ▸ Upload files.
2) Drag all files above (unzipped) → Commit changes.
3) Settings → Pages → Build & deployment: Deploy from a branch (Branch: main, Folder: /(root)) → Save.
4) Visit https://<username>.github.io/<repo-name>/  (use an Incognito tab if you don’t see changes).
